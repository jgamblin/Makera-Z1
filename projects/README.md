# Projects

Each subfolder in this directory represents a single completed (or in-progress) project on the Makera Z1. Every project is self-contained — source design files, G-code / laser paths, photos, and a write-up all live together in one place.

## Starting a New Project

1. Copy the `_template/` folder and rename it to something descriptive:
   ```
   cp -r _template/ my-new-project
   ```
2. Fill out the `README.md` inside your new folder following the template prompts.
3. Add source files (SVG, DXF, F3D, LBRN2, etc.) and final outputs (NC, GCODE) into the same folder.
4. Commit regularly — even mid-project notes are valuable.
